# Wrkstrm Finance

**Systems and tooling for modern finance workflows.**

Wrkstrm Finance builds shared libraries and infrastructure for finance apps, data
pipelines, and trading workflows across the Wrkstrm ecosystem.

### Focus Areas
- **Market data:** ingestion, normalization, and observability.
- **Tooling:** CLIs and automation for research + operations.
- **Reliability:** correctness, resilience, and reproducible builds.
